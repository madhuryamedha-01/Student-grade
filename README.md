# Student Grade Management System

## Project Description

The Student Grade Management System is a simple Java console application that allows users to enter student marks and automatically calculates the total, average, and grade based on predefined grading criteria.

## Features

* Accepts student name and marks as input
* Calculates total marks
* Computes average marks
* Assigns grade based on the average
* Displays the result in the console

## Development Tool

* Eclipse IDE

## How the Program Works

1. The program asks the user to enter the student name.
2. The user enters marks for three subjects.
3. The program calculates the total and average marks.
4. Based on the average, the program assigns a grade (A, B, C, or D).
5. The final result is displayed in the console.

## Example Output

Enter Student Name: Elsa
Enter marks of 3 subjects: 80 85 90
